# 🚀 AI Object Detection using YOLOv8

An AI-powered Object Detection Web Application built using **YOLOv8**, **Python**, **OpenCV**, and **Streamlit**.
The application can detect multiple objects in images and videos with high accuracy through an interactive web interface.

## 🌐 Live Demo

**🔗 Live Application:** https://objectdetection-gsrwdsmtd4tuabd6y3wjz9.streamlit.app/

---

## 📌 Features

* 🖼️ Object Detection on Images
* 🎥 Object Detection on Videos
* 🎯 Adjustable Confidence Threshold
* 📦 Upload Custom YOLO (.pt) Models
* ⚡ Fast Inference with YOLOv8
* 📊 Detection Results with Bounding Boxes & Confidence Scores
* 💻 Clean and Interactive Streamlit Interface

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/Megha8252/Object_Detection.git
```

### Move to the Project Directory

```bash
cd Object_Detection
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app_streamlit.py
```
